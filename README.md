# Even or Odd Verifier

Simple Java console program that checks if a number is even or odd.  

<hr>

## Features
- Input validation: only numbers allowed
- Loops until user decides to stop
- Friendly messages to guide the user

  <hr>

## How to Run
1. Make sure you have [Java JDK](https://www.oracle.com/java/technologies/javase-downloads.html) installed.
2. Clone this repository.
3. Compile the program:
   ```bash
   javac Main.java
   ```
4. Run the program:
   ```bash
   java Main
   ```
